# puik-theme

> puik component theme.

## Installation

```shell
npm i puik
```

## Usage

Use Sass import

```css
@use 'puik/lib/theme/index.scss';
```

Or Use vite/webpack

```javascript
import 'puik/lib/theme/index.css'
```

Or

```html
<link rel="stylesheet" href="https://unpkg.com/puik/lib/theme/index.css" />
```
